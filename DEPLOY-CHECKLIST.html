🚀 GitHub

Create repo on GitHub
Go to github.com/new → name it gem-n-eye → ❌ do NOT check "Initialize with README" → Create
Run push-to-github.sh
Open Terminal → cd into GemNEye folder → run: chmod +x push-to-github.sh && ./push-to-github.sh

⚡ Vercel

Import repo to Vercel
vercel.com/new → Import from GitHub → select gem-n-eye
Get Gemini API key
Free at aistudio.google.com/app/apikey → copy the key
Add environment variables in Vercel
Project Settings → Environment Variables → add all 5:
GEMINI_API_KEY · SUPABASE_URL · SUPABASE_ANON_KEY · SUPABASE_SERVICE_KEY · STRIPE_SECRET_KEY
Deploy on Vercel
Click Deploy — wait for the green checkmark. Copy your live URL (e.g. gem-n-eye.vercel.app)

💳 Stripe

Add Stripe webhook endpoint
Stripe Dashboard → Developers → Webhooks → Add endpoint
URL: https://YOUR-VERCEL-URL.vercel.app/api/webhook
Event: checkout.session.completed
Add STRIPE_WEBHOOK_SECRET to Vercel
Copy the signing secret from the webhook → paste into Vercel env vars → Redeploy
Set Stripe success redirect URLs
In Stripe, edit each payment link's "After payment" redirect:
$9 link → https://YOUR-URL/gem-n-eye-slots.html
$19 link → same URL
(Webhook handles the grant automatically — no URL params needed anymore)
